A powerful plugin to manage and display your links with customizable settings.

== Description ==

WooPop allows you to manage and display various types of links, customize their appearance, and optimize your link page for SEO. Perfect for creating a personalized link hub.

== Installation ==

1. Upload the `WooPop` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to `WooPop` in the admin menu to configure settings.

== Frequently Asked Questions ==

= How do I add a new link? =

Go to the WooPop settings page, navigate to the 'Links' tab, and use the 'Add Link' button to create new links.
